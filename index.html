Panduan Konsolidasi Laporan

Cara mudah menggabungkan file paginasi monitoring kedalam satu visualisasi.

1

Buka Menu Input JSON

Klik menu "Input & Kelola JSON" di bilah sidebar menu untuk menampilkan form penyunting kode terpisah.

2

Tambah Sesuai Halaman

Klik "+ Tambah Kotak" untuk menambah input baru. Paste data JSON per halaman dari API survei Anda ke masing-masing kotak.

3

Proses & Analisis

Klik "Satukan & Buka Dashboard". Sistem otomatis mendeteksi kecocokan email/petugas di setiap halaman dan menyatukannya.

Catatan Teknis: Penggabungan didasarkan pada kecocokan identitas email unik. Apabila Koseka memiliki beban tugas yang tersebar di halaman 1 dan halaman 2 untuk petugas yang sama, beban wilayah tersebut akan diakumulasi di dalam baris tunggal petugas bersangkutan pada tabel dashboard.

Tindakan berhasil diselesaikan.
